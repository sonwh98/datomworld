# OpenCode Keyboard Shortcuts

## Scrolling Messages (Output)
| Action                | Shortcut              |
|-----------------------|-----------------------|
| Page up               | Ctrl+Alt+B / PageUp   |
| Page down             | Ctrl+Alt+F / PageDown |
| Line up               | Ctrl+Alt+Y            |
| Line down             | Ctrl+Alt+E            |
| Half page up          | Ctrl+Alt+U            |
| Half page down        | Ctrl+Alt+D            |
| Jump to top           | Ctrl+G / Home         |
| Jump to bottom        | Ctrl+Alt+G / End      |

| Cycle agent reverse   | Shift+Tab             |
| Interrupt             | Escape                |
| Compact session       | Ctrl+X then C         |
| Rename session        | Ctrl+R                |
| Delete session        | Ctrl+D                |
| Undo last change      | Ctrl+X then U         |
| Redo change           | Ctrl+X then R         |
| Child session first   | Ctrl+X then Down      |
| Next child            | Right                 |
| Previous child        | Left                  |
| Parent session        | Up                    |
| Copy message          | Ctrl+X then Y         |
| Toggle conceal        | Ctrl+X then H         |
